### 添加校区

**请求地址**:
```
    POST     /api/v1/campus/
```

**请求参数**:
```
    name  str  校区名称 非空 唯一 
    info  str  校区描述 非空
```


**成功返回**：
```
{
    "code": 0,
    "msg": "请求成功",
    "data": {
        "id": 4,
        "name": "重庆",
        "info": "重庆校区",
        "create_time": "2018-08-08T09:35:06.547000Z",
        "update_time": "2018-08-08T09:35:06.548000Z"
    },
    "field_name": ""
}
```
